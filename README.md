# Aspectos claves de Git y Git Hub

1. ¿Qué es Git?

Es un software de control de versiones. Es de código abierto, gratuito, robusto y descentralizado. Básicamente es nuestro repositorio local.

2. ¿Qué es Git Hub?

Es una plataforma que permite alojar proyectos utilizando el sistema de versiones de Git. Es una interfaz gráfica en la web de nuestro repositorio remoto.

3. ¿Qué es el staging area?

Área donde se almacenan de manera temporal los archivos preparados que estarán en el commit.
